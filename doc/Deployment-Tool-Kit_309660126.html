DTK : Deployment Tool Kit

The Deployment Tool Kit (DTK) packs a series of features that simplify the configuration cycle of OCE projects.

This tool is done in order to handle the gap in retrieving and deploying SF standard metadata and OCE managed metadata. The mentioned gap is about how to manage package.xml for retrieving and deploying. Instead of maintaining a file with manual inputs the tool is generating the file to retrieve all the metadata components
